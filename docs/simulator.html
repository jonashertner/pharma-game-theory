Scenario simulator

Adjust the probability and severity inputs to see how the four decision-tree branches re-rank under your beliefs. The math is the decision-quality framework; all probabilities are user-controllable. The simulator shows which branch has the highest expected utility for any input combination, and surfaces the threshold at which the recommendation flips.
